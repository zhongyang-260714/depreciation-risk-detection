> 发榜单位：中国青少年发展基金会（中国平安励志计划）  
> 所属领域：人工智能（主擂台：上海徐汇区）

## 一、项目简介

**AI 风口之上，财报之下，隐藏着数百亿美元的虚假利润。**

全球五大科技巨头（Meta、Google、Amazon、Microsoft、Tesla）2024-2025 年度 AI 资本支出合计超过 **2,000 亿美元**。但这些 GPU 集群、数据中心、AI 训练基础设施在会计准则下按 **5-10 年周期**摊销折旧，而 AI 技术迭代周期只有 **1-3 年**。这意味着：资产在账面上"活着"，在市场上"死了"——**数百亿美元的成本被递延，数百亿美元的利润是虚假的**。

本项目构建**全球首个 AI 驱动的"AI泡沫 × 资产折旧错配"系统性风险识别系统**，聚焦美国科创企业，通过多模态数据融合与可解释 AI，识别财报背后的隐形泡沫，预警潜在的全球性金融风险。

### 三重错配
- **时间错配**：技术迭代周期(1-3年) << 会计摊销周期(5-10年)
- **规则错配**：工业时代准则(GAAP) → 数字时代现实(AI资产快速淘汰)
- **激励错配**：AI 军备竞赛 × 业绩美化动机 × 审计师技术盲区

### 全球影响
- 全球养老金、主权基金、散户通过 401(k)、ETF 重仓美股科技股
- 美国科技巨头财报是全球投资者的**信任锚点**
- 折旧陷阱一旦暴露，可能触发**小型全球性金融危机**

## 二、战略定位

### 核心路线：中国视角 → 全球问题 → 美国为典型案例

> **一句话定位："用美国的数据，建全球的算法，防中国的泡沫。"**

本项目回答的是**全球性问题**——AI 资产折旧错配导致的系统性泡沫，但采用**分层策略**：

| 层级 | 内容 | 说明 |
|------|------|------|
| **问题层** | AI 资产折旧错配 → 财报虚假 → 系统性泡沫 | 全球性问题，美国是震中 |
| **案例层** | 美国 AI 巨头（Meta/Google/Microsoft/Amazon/Tesla） | 数据最完整、问题最典型、冲击力最强 |
| **系统层** | 通用算法框架（可适配 GAAP / CAS 6 / IAS 38） | 用美国数据训练，但规则引擎支持多国会计准则 |
| **验证层** | 中国科创板 AI 企业（寒武纪、商汤等） | 证明系统可迁移，问题正在中国发生 |
| **落地层** | 中国证监会、科创板审计、中国投资者 | 回答"为什么中国需要这个" |

详见 `docs/项目战略定位.md` 和 `docs/战略抉择备忘录.md`。

## 六、快速开始

### 1. 环境安装

```bash
# 创建虚拟环境（推荐）
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate    # Windows

# 安装依赖
pip install -r requirements.txt
```

### 2. 数据准备

详见 `docs/data_guide.md`，数据目录（`data/`）需手动下载，不在版本控制内。

### 3. 运行示例

```bash
# 特征工程 + 模型训练（示例）
python -m src.models.train --config config/default.yaml

# 启动Streamlit演示界面
streamlit run src/dashboard/app.py
```
