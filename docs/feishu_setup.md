## 三、创建多维表格（任务看板）

### 表格结构

| 字段 | 类型 | 说明 |
|-----|------|------|
| 任务名称 | 文本 | 具体任务描述 |
| 负责人 | 人员 | @负责人 |
| 阶段 | 单选 | Phase 1/2/3/4 |
| 优先级 | 单选 | 紧急 / 高 / 中 / 低 |
| 开始日期 | 日期 | 计划开始 |
| 截止日期 | 日期 | 计划完成 |
| 状态 | 单选 | 待办 / 进行中 / 审核中 / 已完成 / 阻塞 |
| 完成进度 | 进度 | 0-100% |
| 关联文件 | 附件 | 飞书文档/代码链接 |
| 备注 | 文本 | 遇到的问题或说明 |

### 视图设置

1. **看板视图**：按「状态」分组
2. **甘特图视图**：按时间线展示
3. **我的任务**：按负责人筛选

## 四、设置机器人通知（可选）

### GitHub 自动通知

1. 在飞书群设置 → 添加机器人 → 自定义机器人
2. 复制 Webhook 地址
3. 在 GitHub 仓库设置 → Webhooks → 添加新 Webhook
4. 填写飞书 Webhook URL，选择推送事件

这样每次代码提交都会自动通知到飞书群。

## 六、快捷键 & 技巧

- `Ctrl + K`：快速搜索文档
- `@人名`：在文档中提醒某人
- `//`：在文档中快速插入各种元素（表格、代码块、公式等）
- 文档内评论：选中文字 → 添加评论（适合代码 Review）
