# 今天先做这 5 件事（新手速通版）

> 不用看完所有文档，先完成这 5 步，后面的事一步一步来。

---

## 第 1 步：创建 GitHub 仓库（2分钟）

打开你截图的 GitHub 页面，做 3 个改动：

| 项目 | 改成什么 |
|------|---------|
| Description | 填入：`科创企业资产折旧风险识别算法 — 2026揭榜挂帅 XH-202626` |
| Add .gitignore | 选 **Python** |
| Add README | **关闭（点开关变成 Off）** |

然后点击 **Create repository**。

---

## 第 2 步：安装 Git（只做一次，10分钟）

1. 打开 `https://git-scm.com/download/win`
2. 点击下载，下载完成后双击安装
3. **全部保持默认**，一直点「Next」，最后点「Install」

---

## 第 3 步：上传代码到 GitHub（10分钟）

打开「开始菜单」→ 搜索「Git Bash」→ 打开黑色窗口。

依次复制粘贴下面 6 行命令（每行按回车）：

```bash
cd "/c/Users/Lenovo/Documents/kimi/workspace/depreciation-risk-detection"
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/zhongyang-260714/depreciation-risk-detection.git
git push -u origin main
```

最后一行会弹窗让你登录 GitHub，输入用户名和密码即可。

完成后打开浏览器访问 `https://github.com/zhongyang-260714/depreciation-risk-detection`，应该能看到所有文件。

---

## 第 4 步：安装 Python（只做一次，10分钟）

1. 打开 `https://www.python.org/downloads/`
2. 点击黄色按钮「Download Python 3.12.x」
3. 下载后双击安装
4. **关键：勾选底部「Add Python to PATH」**，然后点 Install Now

验证：打开「开始菜单」→ 搜索「cmd」→ 打开黑色窗口，输入 `python --version`，看到版本号说明成功。

---

## 第 5 步：运行演示界面（立刻看到效果）

打开 cmd 黑色窗口，依次输入：

```cmd
cd C:\Users\Lenovo\Documents\kimi\workspace\depreciation-risk-detection
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run src\dashboard\app.py
```

最后一条命令会显示一个网址（如 `http://localhost:8501`），用浏览器打开它，就能看到交互式的风险识别界面！

输入几个数字（比如总资产 10000、无形资产 2000、研发费用 1500），点「开始分析」，立刻出结果。

> 按 `Ctrl + C` 关闭程序。
