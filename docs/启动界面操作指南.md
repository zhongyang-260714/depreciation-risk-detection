# 启动 Streamlit 界面操作指南

> 适用项目：depreciation-risk-detection（科创企业资产折旧风险识别系统）
> 项目位置：`D:\depreciation-risk-detection`（虚拟环境 `venv\` 已建好，无需重建）
> 核心三步：**进入项目文件夹 → 激活虚拟环境 → 启动 Streamlit**

---

## 0. 概念一句话

- **虚拟环境（venv）**：项目专属的"工具箱"，本项目安装的库都在 `venv/` 文件夹里，与电脑其他项目隔离。**每次运行前必须激活**，激活成功的标志是命令行最前面出现 `(venv)` 字样。
- **Streamlit**：把 Python 脚本变成网页的库。启动命令执行后，浏览器打开 `http://localhost:8501` 即可看到界面。

---

## 1. 三种打开方式

### 方式一：CMD / PowerShell（Windows 终端）

```bat
cd /d D:\depreciation-risk-detection
venv\Scripts\activate
streamlit run src/dashboard/app.py
```

- `cd /d` 里的 `/d` 不能省：它表示"同时切换盘符"，否则从 C 盘切不到 D 盘
- PowerShell 激活若报红色错误，改用：`venv\Scripts\Activate.ps1`
- 若提示"禁止运行脚本"，改用 CMD 或 Git Bash，不要改系统策略

### 方式二：Git Bash

```bash
cd /d/depreciation-risk-detection
source venv/Scripts/activate
streamlit run src/dashboard/app.py
```

与 CMD 的两点区别：

| 区别 | CMD / PowerShell | Git Bash |
|---|---|---|
| 项目路径 | `D:\depreciation-risk-detection`（反斜杠 `\`） | `/d/depreciation-risk-detection`（斜杠 `/`，盘符小写、冒号去掉） |
| 激活命令 | `venv\Scripts\activate` | `source venv/Scripts/activate` |

### 方式三：VS Code 内置终端（日常推荐）

1. VS Code 打开项目文件夹
2. 按 `` Ctrl + ` `` 呼出内置终端
3. 激活（CMD 终端用 `venv\Scripts\activate`；Git Bash 终端用 `source venv/Scripts/activate`）
4. 运行 `streamlit run src/dashboard/app.py`

## 2. 常用操作

| 场景 | 操作 |
|---|---|
| 停止界面 | 终端按 `Ctrl + C` |
| 浏览器页签关了想再看 | 浏览器访问 `http://localhost:8501`（终端保持开着） |
| 修改代码后看效果 | 保存文件，网页右上角点 "Rerun"（或按 `R`），无需重启 |
| 退出虚拟环境 | 输入 `deactivate`，或直接关闭终端窗口 |
| 端口 8501 被占用 | 启动命令加 `--server.port 8502` |
| 换了台电脑 / 重装了环境 | 激活 venv 后执行 `pip install streamlit pandas numpy matplotlib pyyaml` |

## 3. 报错对照表

| 报错关键词 | 原因 | 解决 |
|---|---|---|
| `'streamlit' 不是内部或外部命令` | 没激活虚拟环境 | 先执行激活命令，看到 `(venv)` 再启动 |
| `No module named 'streamlit'` | 激活了别的环境 / 库里没装 | 确认 `(venv)` 前缀后 `pip install streamlit` |
| PowerShell 红色 "禁止运行脚本" | 系统执行策略限制 | 改用 CMD 或 Git Bash |
| `Error: Streamlit requires raw Python` 或端口占用 | 8501 被别的程序占了 | 加 `--server.port 8502` |
| 浏览器没自动弹出 | 正常情况 | 手动访问终端里显示的 `http://localhost:8501` |
| 页面中文显示为方块 | 系统缺中文字体 | Windows 一般自带黑体/雅黑，截图联系排查 |

## 4. 界面里有什么（当前版本）

- **侧边栏「🔍 单公司分析」**：手动输入 5 项财务指标（+可选年报文本），输出 0-100 风险评分、等级徽章、雷达图、因子贡献图、分级预警、评分公式说明
- **侧边栏「📚 案例库」**：展示团队已标注的 5 家美国科技公司（MSFT / NVDA / GOOGL / META / AMD）风险画像，数据来自 `data/annotated/*.json`，答辩可直接演示
- 财务指标填写提示：鼠标悬停输入框右侧的 ❓ 图标，会显示该数据在 10-K 年报的哪张表里找
