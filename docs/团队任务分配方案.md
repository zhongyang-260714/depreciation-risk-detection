## 四、接口定义草案（发给队友）

这是你需要在 7月19日 前完成并给队友的接口定义。先给你一个草案：

### 输入接口：`POST /api/v1/analyze`

```python
from pydantic import BaseModel, Field
from typing import Optional

class FinancialData(BaseModel):
    ticker: str = Field(..., description="股票代码，如 META")
    fiscal_year: int = Field(..., ge=2000, le=2024, description="财年")
    total_assets: float = Field(..., gt=0, description="总资产（百万美元）")
    intangible_assets: float = Field(..., ge=0, description="无形资产（百万美元）")
    rd_expense: float = Field(..., ge=0, description="研发费用（百万美元）")
    revenue: float = Field(..., gt=0, description="营业收入（百万美元）")
    depreciation_amortization: float = Field(..., ge=0, description="折旧摊销（百万美元）")
    net_income: float = Field(..., description="净利润（百万美元）")
    capex: float = Field(..., ge=0, description="资本支出（百万美元）")
    
class TextInput(BaseModel):
    risk_factors_text: Optional[str] = Field(None, description="年报风险因素章节文本")

class AnalyzeRequest(BaseModel):
    financial: FinancialData
    text: Optional[TextInput] = None
```

### 输出接口：`AnalyzeResponse`

```python
class RiskMetrics(BaseModel):
    depreciation_rate: float = Field(..., description="折旧摊销率")
    intangible_ratio: float = Field(..., description="无形资产占比")
    rd_intensity: float = Field(..., description="研发强度")
    asset_turnover: float = Field(..., description="资产周转率")
    capex_to_revenue: float = Field(..., description="CAPEX/收入比")

class RiskSignal(BaseModel):
    category: str = Field(..., description="风险类别")
    description: str = Field(..., description="风险描述")
    severity: str = Field(..., description="严重程度：low/medium/high")

class SHAPExplanation(BaseModel):
    feature: str = Field(..., description="特征名称")
    contribution: float = Field(..., description="对评分的贡献值")
    direction: str = Field(..., description="方向：positive/negative")

class AnalyzeResponse(BaseModel):
    risk_score: float = Field(..., ge=0, le=100, description="风险评分 0-100")
    risk_level: str = Field(..., description="风险等级：low/medium/high")
    key_metrics: RiskMetrics
    risk_signals: list[RiskSignal]
    shap_explanation: list[SHAPExplanation]
    analysis_summary: str = Field(..., description="AI生成的风险摘要")
```
