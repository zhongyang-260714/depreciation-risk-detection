## 二、折旧 vs 摊销：分开还是合并？

### 2.1 定义区别

| 项目 | 折旧（Depreciation） | 摊销（Amortization） |
|------|---------------------|---------------------|
| **对象** | 固定资产（服务器、建筑物、设备） | 无形资产（专利、软件、版权、商誉） |
| **原因** | 物理损耗、技术淘汰 | 法律保护到期、技术替代、合同到期 |
| **寿命** | 基于物理预期寿命（4-30年） | 基于法律保护期或合同期（3-10年） |
| **风险来源** | 技术迭代加速、设备物理损坏 | 专利失效、软件过时、收购失败 |
| **Meta 2023** | $11,178M | $161M |

### 2.2 为什么建议分开？

**对于你的项目，分开更好**：

1. **风险来源完全不同**
   - 折旧风险 = AI技术迭代导致GPU/服务器提前报废
   - 摊销风险 = 专利到期或收购的商誉减值
   - 两个风险的驱动因素不同，不能混为一谈

2. **金额差异巨大**
   - Meta 折旧 $11,178M vs 摊销 $161M（相差 70 倍）
   - 如果合并，摊销的微小波动会被折旧的巨大数字淹没
   - 摊销在Meta几乎不重要，但在其他公司（如制药公司）可能很重要

3. **模型解释性更好**
   - SHAP 解释时，"折旧率过高"和"摊销率异常"是两种不同的风险信号
   - 合并后无法区分是哪种风险

### 2.3 实际操作建议

| 场景 | 处理方式 | 原因 |
|------|---------|------|
| **数据收集** | **分开提取** | 折旧和摊销在现金流量表中通常分开列示 |
| **特征工程** | **主要用折旧**，摊销作为辅助 | 折旧是核心风险，摊销是次要风险 |
| **风险评分** | 折旧占权重80%，摊销占20% | 折旧金额大、风险高 |
| **模型输入** | 两个独立特征 | `depreciation` 和 `amortization` |
| **其他公司** | 如果公司只披露合并的D&A，就合并 | 数据不可得时合并也可以 |

**Meta的特殊情况**：
- 折旧 $11,178M，摊销 $161M
- 摊销只有折旧的 1.4%，几乎可忽略
- 对于Meta，合并也不影响结果，但**分开更规范**

## 三、当前指标体系够不够用？

### 3.1 你的基础指标（12个）

```
总资产、无形资产、研发费用、折旧、摊销、
营业收入、净利润、CAPEX、固定资产原值
```

**评价：够用，但不够精细。**

### 3.2 为什么够用？

这些指标可以计算出**核心风险比率**：
- 折旧率 → 判断折旧是否充分
- 研发强度 → 判断资本化风险
- CAPEX/收入 → 判断扩张速度和折旧压力
- 无形资产占比 → 判断技术淘汰风险

### 3.3 为什么不够精细？（可能导致返工）

| 缺失的精细指标 | 为什么重要 | 返工风险 |
|--------------|-----------|---------|
| **固定资产明细（按类别）** | 服务器vs建筑物vs装修，折旧风险完全不同 | 高：如果只用总固定资产，无法识别服务器折旧滞后 |
| **累计折旧** | 判断资产新旧程度，越旧风险越低 | 中：新资产多=折旧风险高 |
| **R&D资本化 vs 费用化** | 费用化是保守，资本化是激进 | 高：如果公司大量资本化R&D，风险完全不同 |
| **无形资产明细（按类型）** | 商誉 vs 软件 vs 专利，减值风险不同 | 中：商誉减值和专利摊销是两回事 |
| **有效税率** | 用于计算折旧对净利润的实际影响 | 低：可以用估算值 |
| **经营现金流** | 判断公司是否有足够现金覆盖折旧 | 中：现金流紧张时减值风险高 |
| **在建工程** | 判断未来折旧压力 | 中：在建工程转固后折旧大增 |
| **资产减值损失** | 当年实际计提了多少减值 | 高：如果已经减值，说明风险已释放 |
| **员工人数** | 科技公司人均资产 = 资本密集度 | 低：辅助指标 |
| **收入结构** | 广告收入 vs 硬件收入 vs 云服务收入 | 中：不同收入模式对资产依赖不同 |

### 3.4 建议：现在定义完整数据结构，避免返工

**核心原则**：**宁可多收集，不要少收集。** 多收集的数据可以后续不用，但少收集的数据会迫使重写代码。

## 五、其他科技公司都有这些指标吗？

### 5.1 美国上市公司（GAAP）——全部都有

| 指标 | 美国GAAP要求 | 披露位置 |
|------|-------------|---------|
| 总资产 | 强制 | 资产负债表 |
| 固定资产净值 | 强制 | 资产负债表 |
| 固定资产原值 | 强制 | 附注7或8（PP&E Schedule） |
| 累计折旧 | 强制 | 附注7或8 |
| 固定资产明细（按类别） | 强制 | 附注7或8 |
| 在建工程 | 强制 | 附注7或8 |
| 无形资产 | 强制 | 资产负债表 + 附注 |
| 商誉 | 强制 | 资产负债表 |
| 收入 | 强制 | 利润表 |
| 净利润 | 强制 | 利润表 |
| 研发费用 | 强制（如果重大） | 利润表或附注 |
| 折旧费用 | 强制 | 现金流量表 |
| 摊销费用 | 强制（如果重大） | 现金流量表或附注 |
| CAPEX | 强制 | 现金流量表 |
| 使用寿命 | 强制 | 附注1（会计政策） |
| 折旧方法 | 强制 | 附注1（会计政策） |
| 资产减值 | 强制（如果发生） | 利润表 |
| 有效税率 | 强制 | 附注（所得税） |
| 员工人数 | 建议披露 | 管理层讨论 |

**结论**：所有美国上市公司的 10-K 中，**你需要的指标全部都有**。只是有些在利润表主表中，有些在附注中，需要仔细阅读。

### 5.2 中国A股上市公司（CAS）——大部分都有

| 指标 | 中国CAS要求 | 注意 |
|------|------------|------|

| 收入 | 强制 | - |

| 研发费用 | 强制 | 但资本化/费用化披露不够详细 |
| 折旧费用 | 强制 | 现金流量表 |
| 摊销费用 | 强制 | 现金流量表 |
| CAPEX | 强制 | 现金流量表 |
| 使用寿命 | 强制 | 附注 |
| 折旧方法 | 强制 | 附注 |
| 资产减值 | 强制 | 利润表 |

**结论**：A股公司也有这些指标，但**披露质量不如美国**。无形资产明细、R&D资本化细节、固定资产明细等披露不够详细。
