## 📋 第一部分：邀请队友加入 GitHub

### 步骤 1：打开仓库设置

1. 打开浏览器，访问你的仓库地址：
   ```
   https://github.com/zhongyang-260714/depreciation-risk-detection
   ```

2. 点击页面顶部的 **Settings**（设置）标签
   - 如果找不到 Settings，点击仓库名称右侧的 ⚙️ 齿轮图标
   - 或者点击页面右上角的三个点菜单

### 步骤 2：进入成员管理

1. 在左侧侧边栏中，点击 **Manage access**（管理访问权限）
   - 如果看不到，点击 **Collaborators and teams**（协作者和团队）

2. 点击绿色的 **Add people**（添加人员）按钮

### 步骤 3：邀请队友

1. 在弹出的搜索框中，输入队友的 **GitHub 用户名** 或 **注册邮箱**
   - 示例：如果队友的用户名是 `zhangsan123`，直接输入即可
   - 如果不确定用户名，让队友登录 GitHub 后把个人主页链接发给你

2. 选择队友的账号 → 点击 **Add [用户名] to this repository**

3. 选择权限级别（建议选择）：
   - **Write**（写入权限）✅ 推荐 — 队友可以推送代码、创建分支、提交PR
   - **Read**（只读权限）— 队友只能看代码，不能修改
   - **Admin**（管理员）— 队友可以删除仓库（不推荐）

### 步骤 4：通知队友

1. 队友会收到 GitHub 邮件通知，需要点击邮件中的链接 **接受邀请**
   - 如果队友没收到邮件，让他登录 GitHub 后在右上角铃铛图标 🔔 里查看通知

2. 队友接受后，在仓库主页的 Settings → Manage access 里能看到他的名字

### ⚠️ 常见问题

| 问题 | 解决方法 |
|------|---------|
| 找不到队友的用户名 | 让队友登录 GitHub → 点击右上角头像 → 点击 "Your profile" → 浏览器地址栏显示的就是用户名 |
| 邀请发送后队友没收到 | 让队友检查垃圾邮件，或在 GitHub 网站右上角铃铛图标里查看 |
| 权限选错了 | Settings → Manage access → 点击队友名字旁边的齿轮 → 修改权限 |

## 📎 附件：队友可能问的问题

### Q1: GitHub 是什么？
**A**: GitHub 是代码托管平台，相当于"代码的百度网盘"+"版本控制工具"。你可以把它理解成：
- 存放代码的地方（网盘）
- 能看到每次修改了什么的记录（版本控制）
- 多人协作时不会互相覆盖代码（分支管理）

### Q2: 飞书和微信有什么区别？
**A**: 飞书更适合项目协作：
- 可以创建任务看板（像 Trello）
- 文档可以多人实时编辑（像 Google Docs）
- 可以关联日历、设置提醒
- 有机器人自动通知 GitHub 代码更新

### Q3: 我需要学什么东西？
**A**: 队友需要会：
- Git 基本操作（add/commit/push/pull）
- Python 基础（你们项目用 Python）
- Streamlit（前端框架，很简单）
- 如果做后端需要了解 FastAPI

### Q4: 如果我改错了代码怎么办？
**A**: Git 的好处就是可以回退。如果改错了：
```bash
git log                    # 查看历史版本
git checkout 版本号        # 回退到某个版本
```
或者直接在 GitHub 网页上查看历史修改。
